其他方法

Print to PDF Method

For documents with less restrictive settings.

  1. Open the view-only Google Doc
  2. Press Ctrl + P (Windows) or Cmd + P (Mac)
  3. In the printer options, select "Save as PDF"
  4. Choose your desired save location
  5. Click "Save"

Note: This method may be blocked in documents with printing disabled.

Phương Pháp In sang PDF

Cho các tài liệu có cài đặt ít hạn chế hơn.

  1. Mở tài liệu Google Doc chỉ xem
  2. Nhấn Ctrl + P (Windows) hoặc Cmd + P (Mac)
  3. Trong tùy chọn máy in, chọn "Lưu dưới dạng PDF"
  4. Chọn vị trí lưu mong muốn
  5. Nhấp vào "Lưu"

Lưu ý: Phương pháp này có thể bị chặn trong các tài liệu có tính năng in bị vô hiệu hóa.

Méthode d'Impression en PDF

Pour les documents avec des paramètres moins restrictifs.

  1. Ouvrez le Google Doc en lecture seule
  2. Appuyez sur Ctrl + P (Windows) ou Cmd + P (Mac)
  3. Dans les options d'impression, sélectionnez "Enregistrer au format PDF"
  4. Choisissez l'emplacement d'enregistrement souhaité
  5. Cliquez sur "Enregistrer"

Remarque : Cette méthode peut être bloquée dans les documents où l'impression est désactivée.

Método de Impresión a PDF

Para documentos con configuraciones menos restrictivas.

  1. Abra el Google Doc de solo lectura
  2. Presione Ctrl + P (Windows) o Cmd + P (Mac)
  3. En las opciones de impresora, seleccione "Guardar como PDF"
  4. Elija la ubicación de guardado deseada
  5. Haga clic en "Guardar"

Nota: Este método puede estar bloqueado en documentos con la impresión deshabilitada.

打印为 PDF 方法

适用于限制较少的文档。

  1. 打开只读 Google Doc
  2. 按 Ctrl + P（Windows）或 Cmd + P（Mac）
  3. 在打印机选项中，选择"另存为 PDF"
  4. 选择您想要的保存位置
  5. 点击"保存"

注意：在禁用打印功能的文档中，此方法可能被阻止。

Mobile Basic View Method

Uses Google's mobile view to bypass some restrictions.

  1. Open the view-only Google Doc in your browser
  2. In the URL bar, replace "edit" with "mobilebasic"
  3. Press Enter to load the new URL
  4. Right-click and select "Save as" > "Webpage, HTML Only"
  5. Open the HTML file in your browser
  6. Print to PDF (Ctrl/Cmd + P)

Phương Pháp Chế Độ Xem Di Động Cơ Bản

Sử dụng chế độ xem di động của Google để vượt qua một số hạn chế.

  1. Mở tài liệu Google Doc chỉ xem trong trình duyệt của bạn

  5. Mở tệp HTML trong trình duyệt của bạn

Extracting Text from Downloaded Google Docs

After downloading a Google Doc as HTML or PDF, you can extract and copy the text:

  1. For HTML files:
    • Open the HTML file in your browser
    • Select the text you want to copy (click and drag)
    • Copy the selected text (Ctrl/Cmd + C)
    • Paste into your preferred text editor
  2. For PDF files:
    • Open the PDF in a PDF reader that supports text selection
    • Use the text selection tool to select content
    • Copy and paste into your document

This method allows you to extract and use the text content while preserving formatting.

Trích Xuất Văn Bản từ Tài Liệu Google Docs Đã Tải Xuống

Sau khi tải xuống tài liệu Google Doc dưới dạng HTML hoặc PDF, bạn có thể trích xuất và sao chép văn bản:

  1. Đối với tệp HTML:
    • Mở tệp HTML trong trình duyệt của bạn
    • Chọn văn bản bạn muốn sao chép (nhấp và kéo)
    • Sao chép văn bản đã chọn (Ctrl/Cmd + C)
    • Dán vào trình soạn thảo văn bản ưa thích của bạn
  2. Đối với tệp PDF:
    • Mở PDF trong trình đọc PDF hỗ trợ lựa chọn văn bản
    • Sử dụng công cụ chọn văn bản để chọn nội dung
    • Sao chép và dán vào tài liệu của bạn

Phương pháp này cho phép bạn trích xuất và sử dụng nội dung văn bản trong khi vẫn giữ định dạng.

Extraction de Texte à partir de Google Docs Téléchargés

Après avoir téléchargé un Google Doc au format HTML ou PDF, vous pouvez extraire et copier le texte :

  1. Pour les fichiers HTML :
    • Ouvrez le fichier HTML dans votre navigateur
    • Sélectionnez le texte que vous souhaitez copier (cliquez et faites glisser)
    • Copiez le texte sélectionné (Ctrl/Cmd + C)
    • Collez dans votre éditeur de texte préféré
  2. Pour les fichiers PDF :
    • Ouvrez le PDF dans un lecteur de PDF qui prend en charge la sélection de texte
    • Utilisez l'outil de sélection de texte pour sélectionner le contenu
    • Copiez et collez dans votre document

Cette méthode vous permet d'extraire et d'utiliser le contenu textuel tout en préservant la mise en forme.

Extracción de Texto de Google Docs Descargados

Después de descargar un Google Doc como HTML o PDF, puede extraer y copiar el texto:

  1. Para archivos HTML:
    • Abra el archivo HTML en su navegador
    • Seleccione el texto que desea copiar (haga clic y arrastre)
    • Copie el texto seleccionado (Ctrl/Cmd + C)
    • Pegue en su editor de texto preferido
  2. Para archivos PDF:
    • Abra el PDF en un lector de PDF que admita la selección de texto
    • Utilice la herramienta de selección de texto para seleccionar contenido
    • Copie y pegue en su documento

Este método le permite extraer y utilizar el contenido de texto mientras preserva el formato.

从下载的 Google 文档中提取文本

将 Google 文档下载为 HTML 或 PDF 后，您可以提取和复制文本：

  1. 对于 HTML 文件：
    • 在浏览器中打开 HTML 文件
    • 选择您想要复制的文本（点击并拖动）
    • 复制选定的文本（Ctrl/Cmd + C）
    • 粘贴到您首选的文本编辑器中
  2. 对于 PDF 文件：
    • 在支持文本选择的 PDF 阅读器中打开 PDF

Bonus: Detailed Method for Copying Text from View-Only Google Docs
